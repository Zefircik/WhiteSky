# WhiteSky Backend

Pornire:

```bash
npm install
npm start
```

Deschide în browser:

```text
http://localhost:3001
```

Admin:

```text
Email: admin@whitesky.ro
Parola: admin123
```

CSS-ul este separat în `public/assets/css/` pe foldere: `base`, `layout`, `components`, `pages`.
Favoritele se salvează pe cont în `data/database.json`.
